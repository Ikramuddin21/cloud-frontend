## Overview

This project representation static page.

## Technologies Used

- HTML
- Vite
- JavaScript
- Tailwind CSS
- Font Awesome for icons

## Installation

Clone the repository:

```bash
git clone https://github.com/Ikramuddin21/cloud-frontend
cd cloud-frontend
npm install
npm run start
```
